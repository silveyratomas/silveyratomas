## Proyecto destacado

### EmpreAssist
Plataforma SaaS orientada a dueños de pequeños negocios que venden productos.

Funciones principales:
- gestión de stock
- ventas
- pedidos a proveedores
- caja
- empleados y encargados
- reportes
- asistente inteligente

**Tecnologías usadas:**
- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase
- PostgreSQL
